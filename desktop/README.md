# Electron + Vite + React + TypeScipt

## Installation

- Clone
- Run `yarn` or `npm install` to install dependencies.
- That's it.

## How To Use

- `yarn dev` Run development server. Blazing Fast reload on save.
- `yarn buil` to build.
